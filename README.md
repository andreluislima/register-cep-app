<h1 align="center" style="font-weight: bold;">Desafio Técnico - Consulta CEP - Solution SA</h1>
<h3>
      <b>Criar uma aplicação web que permita ao usuário:</b>
</h3>

<p>
    <li>Inserir um <strong>CEP</strong> e buscar automaticamente o endereço utilizando a API <strong>ViaCEP</strong>.</li>
    <li>Preencher dados pessoais (nome e CPF).</li>
    <li>salvar os dados em um banco de dados SQL.</li>
    <li>Gerenciar os dados (listar, editar e salvar alterações).</li>
    <li>Disponibilizar o código em um repositório público no GitHub.</li>
</p>

# 📥 Clonar o projeto

``` bash
git clone https://github.com/<seu-usuario>/cep-register.git
cd cep-register
```

## Como Rodar o projeto
``` bash
cd backend
```

Execute o projeto:

``` bash
mvn spring-boot:run
```

Ou compile e execute o `.jar`:

``` bash
mvn clean install
java -jar target/nome-do-projeto.jar
```

O backend ficará disponível em:

    http://localhost:8080

------------------------------------------------------------------------

# 💻 Executando o Frontend

Entre na pasta do frontend:

``` bash
cd frontend
```

Instale as dependências:

``` bash
npm install
```

Execute o projeto:

``` bash
npm run dev
```

O frontend ficará disponível em:

    http://localhost:5173
